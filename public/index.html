Space Shooter Game

This repository contains a desktop Java/LWJGL game. Vercel cannot execute the Java runtime directly, so this page is the deployable landing page for the project.

Use the download link below to get the runnable jar, or run the game locally with:

./gradlew lwjgl3:run (macOS/Linux) or gradlew.bat lwjgl3:run (Windows)

Download game jar

Download the built desktop launcher from this deployment.

java gamedevelopment-1.0.0.jar

Run locally

To play the game on your machine, use the Gradle desktop task.
